# 🌿 E-Route Frontend

An intelligent eco-friendly route planning web application built with **React**, **Vite**, and **Tailwind CSS**. The application helps users discover cleaner, healthier, and safer travel routes by integrating environmental data such as Air Quality Index (AQI), weather conditions, and route recommendations.

## 🚀 Features

- 🔐 User Authentication (JWT)
- 🗺️ Interactive Route Planning
- 🌍 Real-time Air Quality (AQI) Display
- 🌤️ Weather Information
- 📍 Source & Destination Search
- 🌱 Eco Score Visualization
- 📊 Dashboard with Route Analytics
- 📱 Responsive UI
- 🌙 Modern Clean Interface

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- Axios
- React Router
- Leaflet / OpenStreetMap
- REST API Integration

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/RAGINI-06/e-route-frontend.git
```

Navigate to the project:

```bash
cd e-route-frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 🔗 Backend Repository

The frontend communicates with the E-Route Spring Boot backend using REST APIs.

Backend Features:
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- Route Recommendation APIs
- AQI & Weather APIs
